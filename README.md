# Detecção de placas.  

Este projeto utiliza algoritmos de visão computacional para a detecção de placas de veiculos em estabelecimentos residenciais ou comerciais. 

### Linguagem de programação utilizada:  
* Python  

### Tecnologias utilizadas:
* OpenCV
